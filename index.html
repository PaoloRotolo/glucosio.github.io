What is Glucosio?

Glucosio is a open source project focus on creating open source tools for Diabetes. We believe with open tools people with Diabetes can improve their health outcomes.

Unlike other apps for Diabetes we intend to create a feature rich experience with third-party integrations, tips and the ability to support diabetes research.

Free and Open Source

We believe in the power of free and open source software. All of the project source code is available on Github

Want to get involved? Feel free to send us a pull request!
